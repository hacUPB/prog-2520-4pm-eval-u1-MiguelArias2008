## Ejercicio 1

1. Se requiere un algoritmo para obtener la suma de diez cantidades, que se leen de teclado, mediante la utilizacion de un ciclo **While**. realice el diagrama de flujo y el pseudocodigo.

# Pseudocodigo

```
Inicio 
    1 = 0
    Suma = 0
        While i <= 10
            leer cantidad
            i = i + 1
            suma = suma + cantidad
        Fin while
    Mostrar suma
Fin
```
